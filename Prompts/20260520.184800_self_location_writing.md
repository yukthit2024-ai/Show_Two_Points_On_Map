# Self GPS Location Tracking File Writer

## Accomplishments
Successfully implemented automated background tracking output to record the user's own live GPS location directly to the active sessions directory whenever a coordinate change is captured by the hardware services.

## Technical Details

1. **Location Update Lifecycle Integration**:
   - Monitored live system updates inside the running foreground `LocationService.java`.
   - Intercepted the Fused Location Callback trigger `onLocationUpdated(Location location)`.

2. **Clean Filename Generation**:
   - Fetches the active profile display name from `MatrixClient`.
   - Cleans the string to resolve custom characters and symbols:
     - Strips `@` prefixes and split domain qualifiers.
     - Formats clean alphanumeric file markers: `cleanSender`

3. **Session Folder Output**:
   - Created the automatic self location writer `writeSelfLocationToSessions()`.
   - Formats the live self GPS coordinate data into the standard template:
     - Format: `DisplayName|Latitude|Longitude|Color` (sets static primary color `#1976D2`)
   - Safely saves/overwrites the data directly to `/sdcard/Vypeensoft/Friends_Location_Tracker/sessions/<cleanSender>.txt`.

4. **Successful Gradle Verification**:
   - Compiling successfully: **BUILD SUCCESSFUL**!
