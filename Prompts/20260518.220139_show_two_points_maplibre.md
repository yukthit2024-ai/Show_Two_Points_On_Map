# Implementation Plan - Showing Two Random Points on MapLibre

This plan details the steps to modify the workspace to load a MapLibre map showing two random points separated by approximately 1 kilometer (one red, one green), and completely purge all other irrelevant logic from the project source code.

## 1. Clean Up workspace Files
We will completely purge the unnecessary activities, services, adapters, models, utilities, and resources.
Files to be deleted:
- All secondary Activities: `AboutActivity.java`, `GroupsRoomsActivity.java`, `HelpActivity.java`, `MapSettingsActivity.java`, `MatrixCredentialsActivity.java`
- Adapter directory: `adapter/`
- Model directory: `model/`
- Network directory: `network/`
- Service directory: `service/`
- Util directory: `util/` (excluding any necessary helpers if needed, but we don't need any since we will implement everything directly in MainActivity)

## 2. Simplify `AndroidManifest.xml`
We will rewrite `AndroidManifest.xml` to keep only the `MainActivity` declaration and basic permissions (`INTERNET` and `ACCESS_NETWORK_STATE`), removing all other activities, services, location permissions, and broadcast receivers.

## 3. Simplify `activity_main.xml`
We will rewrite `app/src/main/res/layout/activity_main.xml` to contain only a fullscreen `org.maplibre.android.maps.MapView`.

## 4. Rewrite `MainActivity.java`
We will rewrite `MainActivity.java` to perform only the following:
1. Initialize MapLibre.
2. Setup the MapView and get the map asynchronously.
3. Once the map is ready, load a clean styling sheet (`https://tiles.openfreemap.org/styles/liberty`).
4. Generate a random base location (e.g., Cochin, Kerala: `9.9312`, `76.2673`).
5. Calculate a second coordinate exactly 1.0 km away at a random bearing angle ($\theta$) using:
   - $\Delta \text{Lat} = (\text{distance} / 111.12) \times \cos(\theta)$
   - $\Delta \text{Lon} = ((\text{distance} / 111.12) \times \sin(\theta)) / \cos(\text{baseLat})$
6. Programmatically create high-quality Red and Green marker images (anti-aliased circles with border and shadow) and add them to the map style.
7. Initialize a `SymbolManager` to add a red marker at Point A and a green marker at Point B.
8. Center and zoom the camera perfectly over the two points.
9. Keep only the lifecycle delegates for the `MapView`.

## Technical Calculations for Coordinates
Given:
- $\text{Base Coordinate} = (9.93123, 76.26730)$
- $\text{Target Distance} = 1.0 \text{ km}$
- $\text{Latitude scaling} = 111.12 \text{ km/degree}$
- $\text{Longitude scaling} = 111.12 \times \cos(\text{baseLat}) \text{ km/degree}$

We will choose:
- **Red Marker**: `LatLng(9.93123, 76.26730)`
- **Green Marker**: `LatLng(9.93123 + deltaLat, 76.26730 + deltaLon)`
