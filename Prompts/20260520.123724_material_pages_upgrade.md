# Standard Android Material Pages & Dialogs Upgrade

## Accomplishments
Successfully upgraded all settings and configuration pages (Settings, Matrix Credentials, Matrix Rooms Manager, Add Room, Help, and About) to fully native Android Material Design components and layouts.

## Material Architecture Upgrades

1. **Standard Material Dialogs (`MaterialAlertDialogBuilder`)**:
   - Replaced all basic platform `android.app.AlertDialog` elements with native standard `com.google.android.material.dialog.MaterialAlertDialogBuilder`.
   - Restructured dialog headers, button style hierarchies, and window configurations to follow standard Material styling guidelines.

2. **Advanced Material Form Outlines (`TextInputLayout` & `TextInputEditText`)**:
   - Replaced all raw EditText views inside the configuration dialogs with standard `TextInputLayout` containers and `TextInputEditText` inputs.
   - Designed with standard boxed outlines (`BOX_BACKGROUND_OUTLINE`) and an accent stroke in the cobalt primary color (`#1976D2`).
   - Integrated floating labels and placeholder inputs matching standard Material themes.
   - Built-in secure password visibility toggling (`app:passwordToggleEnabled`) for the credentials dialog.

3. **Material Rounded Buttons (`MaterialButton`)**:
   - Replaced the simple green button inside the Matrix Rooms Manager with a highly polished cobalt `com.google.android.material.button.MaterialButton`.
   - Set custom corner rounding (`8dp`), custom margin guidelines, and standardized text bounds to keep dialog options cleanly aligned.
