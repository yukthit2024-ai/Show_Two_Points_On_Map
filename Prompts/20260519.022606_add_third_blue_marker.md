# User Prompt
```
Add a third blue marker which is between 500 meter and 1000 meter from red and green respectively
```

# Implementation Plan - Adding a Third Blue Marker

This plan outlines the mathematical and programming steps to add a third (Blue) marker that stays between 500 and 1000 meters from both the Red and Green markers.

## Mathematical Formulation
1. Let the distance between Red and Green be exactly 1000 meters.
2. The midpoint $M$ of Red and Green is exactly 500 meters from both points.
3. If we offset a point perpendicular to the line connecting Red and Green from $M$ by a distance $h$, the distance to both Red and Green will be $\sqrt{500^2 + h^2}$.
4. We choose $h = 600$ meters.
   * Distance to Red = $\sqrt{500^2 + 600^2} = \sqrt{610,000} \approx 781$ meters.
   * Distance to Green = $\sqrt{500^2 + 600^2} = \sqrt{610,000} \approx 781$ meters.
   * This is perfectly and stably between 500 meters and 1000 meters!

---

## Technical Approach

### 1. Vector Offset Calculation at Startup
In `MainActivity.java`, after computing `redLatLng` and `greenLatLng`:
```java
double midLat = baseLat + deltaLat / 2.0;
double midLon = baseLon + deltaLon / 2.0;

// Offset perpendicular by 600 meters (0.6 km)
double perpDistanceKm = 0.6;
double perpAngle = angle + Math.PI / 2.0; // Rotate by 90 degrees
double perpLat = (perpDistanceKm / 111.12) * Math.cos(perpAngle);
double perpLon = ((perpDistanceKm / 111.12) * Math.sin(perpAngle)) / Math.cos(Math.toRadians(midLat));

blueLatLng = new LatLng(midLat + perpLat, midLon + perpLon);
```

### 2. Map Marker Setup
1. Create a `blueIcon` from a programmatically generated teardrop pin vector bitmap with Blue color (`#2196F3`).
2. Add the `blueMarker` to the map.
3. Make sure the initial camera centers on the midpoint of the three points (which is close to the Red-Green midpoint).

### 3. Movement Loop Integration
In the loop:
1. Shift `blueLatLng` by the same translation vector `(deltaLat, deltaLon)` as the Red and Green points.
2. Call `blueMarker.setPosition(blueLatLng)` to update its position on the map.
3. Since all three points translate by the exact same vector, their relative distances (Red-Green = 1000m, Blue-Red = 781m, Blue-Green = 781m) remain perfectly constant!
