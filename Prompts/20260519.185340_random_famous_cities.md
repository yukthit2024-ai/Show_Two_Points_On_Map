# Implementation Plan - Startup Random Famous City Generator

We will implement a startup randomization feature in `MainActivity.java` that initializes the simulation around a randomly chosen iconic world city on every app launch.

## 1. Key Objectives
- Define a high-fidelity dataset of iconic cities worldwide with accurate GPS coordinates.
- Select a city uniformly at random at startup.
- Center the map view and deploy the tracking markers around the chosen city.
- Display a sleek Toast notification to inform the user which global city the simulation has loaded.
