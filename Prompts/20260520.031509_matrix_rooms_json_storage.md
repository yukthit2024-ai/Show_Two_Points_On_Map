# Matrix Rooms JSON Storage

## Accomplishments
Successfully implemented automated loading and saving of Matrix Rooms from/to `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/rooms_settings.json`.

## Technical Details

1. **Storage Path**:
   - `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/rooms_settings.json`
   - Dynamically falls back to standard external storage directory API if environment variations exist.

2. **JSON Handling**:
   - Built a custom parser utilizing standard `org.json.JSONObject` to parse the outer JSON object and retrieve the `"matrix_rooms"` string.
   - Replaces single quotes (`'`) inside the rooms array string with standard double quotes (`"`) automatically, ensuring compatibility with strict JSON serializers and enabling 100% successful deserialization into GroupRoom collections via GSON.
   - When rooms are created, toggled (for active selection), or deleted via the UI list, they are automatically saved back to this JSON configuration file dynamically.
