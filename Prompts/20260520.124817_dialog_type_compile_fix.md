# Dialog Type Compile Fix

## Accomplishments
Successfully resolved a Java compilation mismatch (`incompatible types: androidx.appcompat.app.AlertDialog cannot be converted to android.app.AlertDialog`) caused by assigning the result of `MaterialAlertDialogBuilder.create()` to a legacy `android.app.AlertDialog` variable.

## Technical Details

1. **Clean Dialog Variable Mapping**:
   - Refactored `android.app.AlertDialog dialog = builder.create();` inside `showMatrixRoomsDialog()` to use standard `androidx.appcompat.app.AlertDialog`.

2. **Method Signature Integration**:
   - Updated the signature of `showAddRoomDialog()` to receive standard `androidx.appcompat.app.AlertDialog parentDialog` matching the correct Type hierarchy perfectly.
