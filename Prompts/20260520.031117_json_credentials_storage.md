# Matrix Credentials JSON Loader & Storage

## Accomplishments
Successfully implemented automated loading and saving of the Matrix credentials from/to the external JSON config file.

## Technical Details

1. **Storage Path**:
   - `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/matrix_credentails.json`
   - Dynamically falls back to standard external storage directory API if environment variations exist.

2. **JSON Handling**:
   - Built a robust parser utilizing standard `org.json.JSONObject` to parse homeserver URLs, passwords, display names, cached tokens, usernames, and polling intervals.
   - Values are pre-populated into the Matrix Credentials configuration screen immediately.
   - When a user changes credentials and saves them via the UI, a pretty-printed, structured JSON configuration is written directly back to the specified file to keep preferences fully in sync.

3. **Startup Auto-Sync**:
   - The application automatically calls `loadMatrixCredentialsOnStartup()` inside `onCreate()` when launching, guaranteeing that if the configuration file is present on the storage, background syncing and room polling parameters initialize instantly without requiring any manual user interaction.
