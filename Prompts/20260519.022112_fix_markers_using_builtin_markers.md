# Implementation Plan - Displaying Markers Using Built-In Marker API

This plan outlines a highly robust strategy to guarantee marker visibility by transitioning from the external `SymbolManager` plugin (which can suffer from style layout conflicts and asset scaling issues) to MapLibre's built-in `Marker` and `Icon` APIs.

## Why Built-In Markers Guarantee Success
1. **Direct Map Integration**: MapLibre's standard `addMarker()` API injects the markers directly into the map engine pipeline. It is not dependent on the Mapbox/MapLibre GL annotation plugin, which eliminates layout timing bugs or overlap-collision issues.
2. **Self-Contained Vector Pins**: We will programmatically render custom classic teardrop-shaped vector pins. This completely removes any dependency on loading or tinting external PNG assets, resolving any resource loading bugs or color tint filters.
3. **Instant UI Synchronization**: Calling `Marker.setPosition(latLng)` updates the marker's coordinate on the screen instantly, making the loop logic clean and lightweight.

---

## Technical Approach

### 1. Custom Teardrop Pin Vector Generation
We will programmatically draw a classic teardrop pin on a `128x128` canvas. The pin will feature:
* A soft drop shadow under the pointer tip
* A crisp white outer border
* A solid colored core (Crimson Red / Vibrant Green)
* A glowing white central target dot for a highly polished UI.

### 2. Map Setup Sequence
Once the map style finishes loading:
1. Create a `redIcon` and `greenIcon` from our programmatically generated teardrop pin bitmaps using `IconFactory`.
2. Add the markers to the map using `mapLibreMap.addMarker(new MarkerOptions()...)`.
3. Center the camera once at the Cochin midpoint using `moveCamera(newLatLngZoom(midpoint, 14.5))`.
4. Trigger the repeating 1-second movement loop.

### 3. Loop Position Updates
Inside the movement runnable:
1. Apply the 1-meter random walk offset to the coordinates.
2. Call `redMarker.setPosition(redLatLng)` and `greenMarker.setPosition(greenLatLng)` to move them instantly.
